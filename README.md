# 🍋 The Daily Lemon
**AI Data Centre Fitout Intelligence — Australia**

Automated daily agent that scrapes government tender portals and industry news
for AI data centre fitout jobs, then generates a one-page PDF brief and deploys
a narrative website to Netlify.

---

## What It Does

Every morning at **7:15 AM AEST**, the agent:
1. Scrapes AusTender, QTenders, NSW eTendering, EstimateOne, and news RSS feeds
2. Filters for AI data centre fitout opportunities
3. Prioritises **Central Queensland** and **Casino NSW** leads
4. Flags trade matches (CNC joinery, bulkheads, access floors, ceilings, partitions)
5. Generates a one-page PDF: `site/reports/daily-lemon-YYYY-MM-DD.pdf`
6. Updates `site/data.json` for the website
7. Commits changes and deploys to Netlify automatically

## Setup — 5 Steps

### 1. Clone and install
```bash
git clone https://github.com/YOUR_USERNAME/daily-lemon.git
cd daily-lemon
pip install -r requirements.txt
```

### 2. Test locally
```bash
python agent.py
# Opens site/reports/daily-lemon-YYYY-MM-DD.pdf
```

### 3. Push to GitHub
```bash
git remote set-url origin https://github.com/YOUR_USERNAME/daily-lemon.git
git push -u origin main
```

### 4. Connect Netlify
1. Go to [app.netlify.com](https://app.netlify.com) → New site from Git
2. Pick your GitHub repo → **Build command: leave empty** → **Publish directory: `site`**
3. Deploy. Copy your **Site ID** from Site Settings → General

### 5. Add GitHub Secrets
In your GitHub repo → Settings → Secrets → Actions:

| Secret name | Where to get it |
|---|---|
| `NETLIFY_AUTH_TOKEN` | Netlify → User Settings → Applications → Personal access tokens |
| `NETLIFY_SITE_ID` | Netlify → Site Settings → General → Site ID |

That's it. GitHub Actions runs the agent daily at 7:15 AM AEST and Netlify serves the result.

## Priority Areas

The agent gives special prominence (coloured banners, top of page) to:
- **Central Queensland** — Rockhampton, Gladstone, Mackay, Emerald, Yeppoon, Biloela
- **Casino NSW** — Casino, Richmond Valley, Lismore, Northern Rivers

These areas are flagged because they have strong incoming AI infrastructure demand
but limited specialist fitout subcontractors — making them high-value for a housing
construction company pivoting into data centre work.

---

## Trade Match Flags

The agent flags any lead mentioning: CNC · joinery · bulkhead · access floors ·
raised floors · partitions · ceilings · carpentry · cabinetry · NOC rooms ·
doors & frames · epoxy flooring · interior fitout

These overlap directly with housing construction capabilities.
